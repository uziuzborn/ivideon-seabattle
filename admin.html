Публичное объявление

Крупный текст на табло, который видит команда всегда. Например: «На этой неделе стреляет Иванов!» или девиз турнира.

Порог допуска к розыгрышу

Стреляют только те, кто взял планку по звонкам. Среди прошедших — два выстрела: лидеру по звонкам и лидеру по выручке. Порог и правило видны команде на табло. Само значение звонков ты берёшь из PowerBI и вводишь при выстреле — игра сверит с порогом автоматически.

Турниры

Можно вести несколько турниров одновременно — например, отдельно для отдела продаж ПВЗ и для офисных подчинённых. Переключайся между ними здесь. Активный турнир показывается на табло.
При создании попросит подтверждение, чтобы не нажать случайно.

Сайт-табло для команды (GitHub Pages)

Собирает файл index.html — статичное публичное табло с текущим состоянием игры. В файле нет расстановки живых кораблей, секретных призов и пароля — только то, что уже открыто выстрелами. Его безопасно выкладывать в открытый интернет. Обновляй после каждого еженедельного выстрела: собрал → загрузил в GitHub → ссылка у команды показывает свежее табло.

Сохранение и перенос

Всё сохраняется в этом браузере автоматически. Для резервной копии или переноса — скачай файл и потом загрузи. Скачивается сразу всё (все турниры).

Пароль админки

Защищает от случайного клика и любопытных глаз при показе экрана. Это не криптозащита: тот, кто откроет исходник файла, пароль обойдёт.

Опасная зона

Сбросить текущий турнир (обнулить выстрелы и расстановку) или удалить его целиком. Оба действия — с двойным подтверждением.
