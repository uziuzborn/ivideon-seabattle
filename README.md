# Ivideon · Морской бой

Внутренняя площадка геймификации командных результатов.

- `index.html` — публичное табло активных игр.
- `admin.html` — панель администратора и Owner.
- `editor.html` — редактор выбранной игры.
- `config.js` — публичная конфигурация Supabase.

Инструкция доступна внутри панели по кнопке **«? Инструкция»**. В ней также описан рекомендуемый публичный сценарий проведения выстрела: победитель называет координату при команде, а администратор открывает клетку на общем экране.
